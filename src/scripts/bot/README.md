# Структура модулей Telegram бота

Этот каталог содержит модульную структуру для Telegram бота Post Stock Pro.

## Структура

```
bot/
├── commands/          # Обработчики команд
│   ├── start.ts      # Команда /start с улучшенным онбордингом
│   ├── profile.ts    # Команда /me с переключателем ролей
│   ├── subscription.ts # Команда /subscribe с описанием тарифов
│   └── referrals.ts  # Команда /referrals с кнопкой поделиться
├── keyboards/         # Клавиатуры (Reply и Inline)
│   ├── main-menu.ts  # Главное меню (компактное, 2-3 ряда)
│   ├── profile.ts    # Клавиатура профиля
│   ├── subscription.ts # Клавиатура подписки
│   ├── referrals.ts  # Клавиатура рефералов
│   └── onboarding.ts # Клавиатура онбординга
├── middlewares/       # Middleware для проверки авторизации
│   └── auth.ts       # Проверка авторизации и ролей
├── scenes/           # Сцены (визарды) для сложных процессов
│   └── post-generation.ts # Обработка генерации постов
└── utils/            # Утилиты
    └── error-handler.ts # Глобальный обработчик ошибок
```

## Реализованные улучшения

### ✅ Новый /start с баннером
- Приветственное сообщение с описанием ценностей бота
- Inline-кнопки для выбора роли
- Поддержка реферальных ссылок

### ✅ Улучшенный профиль
- Карточка пользователя с полной информацией
- Переключатель ролей (Продавец ↔ Клиент)
- Статистика (рефералы, созданные форматы)
- Inline-кнопки для навигации

### ✅ Меню подписки
- Описание тарифного плана PRO
- Список возможностей
- Информация о текущей подписке
- Кнопка покупки через Telegram Stars

### ✅ Реферальная программа
- Статистика (переходы, регистрации, оплатившие)
- Кнопка "Поделиться ссылкой" (switch_inline_query)
- Кнопка "Копировать ссылку"

### ✅ Переход на Inline-кнопки
- Главное меню осталось Reply-клавиатурой (компактное, 2-3 ряда)
- Все вложенные меню используют Inline-кнопки
- Кнопка "Назад" во всех меню

### ✅ Обработка ошибок
- Глобальный обработчик ошибок
- Понятные сообщения пользователю
- Логирование для админа

## Использование

Главный файл бота (`telegram-bot-polling.ts`) импортирует все модули и использует их для обработки команд и callback.
