# Документация проекта

Эта папка содержит важные инструкции и документацию для проекта Telegram Seller MVP.

## Содержимое

### Развертывание
- **LOCAL_DEPLOYMENT.md** - Подробная инструкция по локальному развертыванию приложения и бота
- **SERVER_DEPLOYMENT.md** - Инструкция по развертыванию на production сервере

### Документация проекта
- **FEATURES.md** - Полное описание всех существующих функций приложения
- **PROJECT_REQUIREMENTS.md** - Полные требования к проекту, архитектура, функционал
- **TZ_MVP_DORABOTKA.md** - Техническое задание на доработку MVP
- **ADMIN_GUIDE.md** - Руководство по созданию администратора и работе с админ-панелью

### Тестирование
- **TESTING_GUIDE.md** - Полный гайд по тестированию всех функций приложения
- **TESTING_EMPTY_STATE.md** - Инструкция по тестированию пустого состояния приложения
- **PRODUCTION_CHECKLIST.md** - Чек-лист подготовки к production запуску

### Безопасность
- **SECURITY_INFO.md** - Информация о безопасности, защите API endpoints
- **SECURITY_AUDIT_REPORT.md** - Отчет об аудите безопасности и исправленных проблемах

### Утилиты
- **SQL_COMMANDS.md** - Полезные SQL команды для работы с базой данных

## Структура документации

Все документы организованы по категориям для удобного поиска нужной информации.
